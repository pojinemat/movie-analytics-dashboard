Drop your Power BI source file here, e.g. MovieDashboard.pbix

This folder is just for keeping the .pbix source under version control.
It is NOT used by the live website — the live charts are rendered by
script.js from data/movies.json. The .pbix here is only useful to people
who open it in Power BI Desktop directly.

If you publish a Power BI report to the web, embed the resulting iframe
in index.html instead (see the README for instructions).
